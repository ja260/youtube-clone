# YouTube Clone (Spring Boot + Thymeleaf)

🔴 **Live Demo:** https://youtube-clone-mxis.onrender.com/

A simple YouTube-like UI built using **Spring Boot (Java)** and **Thymeleaf**.
It supports dynamic video playback using URL parameters and a recommended list.

## Features
- Home page UI (topbar + sidebar + player + recommendations)
- Dynamic video switching using `/?v=INDEX`
- Supports YouTube embed URLs
- Clean static assets structure (CSS/JS)

## Tech Stack
- Java 21
- Spring Boot
- Thymeleaf
- HTML/CSS/JS

## Run Locally
```bash
# In project folder
./mvnw spring-boot:run
